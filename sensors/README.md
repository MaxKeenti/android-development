# sensors/ — Sensors & Hardware APIs

Tasks in this topic teach how to read device sensors, request the permissions they require, and react to hardware events in real time.

## Tasks

| Task | Primary Concept | Notes |
|------|----------------|-------|
| [sensor-management](sensor-management/) | SensorManager, SensorEventListener, runtime permissions | [NOTES.md](sensor-management/NOTES.md) |

## When to Add to sensors/

A new task belongs in `sensors/` when the primary concept the teacher is introducing is reading hardware sensor data (accelerometer, gyroscope, barometer, etc.) or requesting the permissions that protect that data. See the [Topic Decision Gate](../README.md#topic-decision-gate) in the root README for the full rule.
